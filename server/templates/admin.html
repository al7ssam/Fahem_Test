نمط مذاكرة (Study)

نفس صيغة المباشر + حقل studyBody كنص مذاكرة للسؤال.

[
  {
    "prompt": "ما وظيفة الكلوروفيل في النبات؟",
    "options": ["نقل الماء", "امتصاص الضوء", "تخزين النشا", "حماية الجذور"],
    "correctIndex": 1,
    "studyBody": "الكلوروفيل صبغة خضراء تمتص الضوء لبدء عملية البناء الضوئي.",
    "difficulty": "medium"
  }
]

يُقبل أيضًا correct_index بدل correctIndex و study_body بدل studyBody.
